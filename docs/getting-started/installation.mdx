Before you can start using Kohl's Admin, it needs to be installed in your Roblox game. This section will guide you through the initial setup process.

In this guide, we will use the Kohl's Admin (KA) Official Model. You can also explore Kohl's Admin features in the Test Place.

## Prerequisites

Make sure you have Roblox Studio installed on your environment.

- **Roblox Studio**: Download and install [Roblox Studio](https://www.roblox.com/create)

## Installation

:::danger
Be aware of models impersonating Kohl's Admin. The model should only be made by [@Scripth](https://www.roblox.com/users/44391621/profile).
Unofficial models may contain unwanted content such as, but not limited to: viruses, backdoors, unwanted admins.
:::

### Step 1: Acquire the KA Official Loader Model

1. Go to the [KA Official Loader](https://create.roblox.com/store/asset/172732271/Kohls-Admin-Beta) page.
2. Click the **Get Model** button to add the model to your inventory.

### Step 2: Open Roblox Studio

1. Launch **Roblox Studio** on your computer.
2. Open the experience you want to install the KA Loader in or create a new experience.

### Step 3: Import the KA Official Loader Model

1. In Roblox Studio, open the **Toolbox** by selecting `View > Toolbox` from the top menu.
2. In the Toolbox, click on the **Inventory** tab and search for **Kohl's Admin [BETA]** and click on the model.
3. Click OK on the script warning prompt.
4. Once the KA Loader model is in place, save your changes by selecting `File > Publish to Roblox`.
5. You may also participate in Kohl's Game Hub, by going clicking on **Game Settings**, then going to `Security > Allow HTTP Requests`. This is completely optional.

### Step 4: Verify Installation

1. Open and play the experience that you've imported the model into.
2. Test some basic commands to make sure the admin commands are functioning as expected.
